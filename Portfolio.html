Portfolio Projects

Library Management system using SQL: In this project, we were tasked to integrate a MySQL database with Python to develop an advanced Library Management System. This command-line-based application is designed to streamline the management of books and resources within a library. Our goal was to create a robust system that allows users to browse, borrow, return, and explore a collection of books while demonstrating proficiency in database integration, SQL, and Python.
E-commerce API: The goal of this project was to build a user-friendly platform where customers can effortlessly browse products, add them to their shopping carts, and place orders. Simultaneously, administrators should have tools to manage product inventory, track orders, and ensure a seamless shopping experience. To tackle this challenge, I used the powers learned from Python, along with the new knowledge of Flask and Flask-SQLAlchemy, to accomplish this goal.
